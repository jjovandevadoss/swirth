Field Mapping Examples

Common Field Path Examples:

Source Path Description
patient.id Patient ID
patient.name.family_name Patient last name (nested field)
observations[0].value First observation value (array index)
observations[*].value All observation values (array iteration)
orders[0].universal_service_id Test/order name

Tips:

  • Use . (dot) to access nested fields: patient.name.given_name
  • Use [0] to get a specific array item: observations[0]
  • Use [*] to map all array items: observations[*].value
  • Click the icon to browse available fields
